# Less总结
* ###### 可以重复声明变量，但是和css一样，最后一个声明的变量会作为最终生效的样式

* ###### 变量是预先加载的，可以不预先声明，有点跟var相识，先使用在声明

* ###### 在分号（:）之前所有用变量的地方，要用大括号括起来，分好之后不用

* ###### 减号（-）不要和变量挨在一起，否则会报错，减号两边最好打上空格隔开

* ###### less中可以嵌入js,因为本身就是有JS所写，使用~“`~"字符@{变量}字符"`”的结构
